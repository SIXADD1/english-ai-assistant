# 英语 AI 助手 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位

专为四六级备考用户打造的英语AI 辅导平台。全程贴合四六级官方阅卷规则和评分标准，覆盖「素材学习 → 专项训练 → 在线写作 → AI 批改 → 复盘提升」闭环式写作备考体系。

### 1.2 目标用户

- 全国大学英语四级、六级备考考生
- 写作薄弱、不清楚官方评分规则的考生
- 需要反复练习 + 即时反馈的考生

### 1.3 核心能力

| 能力 | 实现方式 |
|------|---------|
| AI 作文批改 | DeepSeek API，按四级/六级不同标准打分 |
| 专项训练 | 审题构思、素材运用、开放式结文、文体格式四大板块 |
| 素材库 | 高频主题词汇、高分句式、衔接逻辑、万能开头结尾 |
| 管理后台 | 管理员可增删改查素材、题目、训练题，管理系统配置 |

## 4. AI 批改说明

### 4.1 评分标准

| 档位 | 分数 | 标准 |
|------|------|------|
| 优秀 | 11-13 分 | 切题，表达思想清楚，文字通顺，连贯性好 |
| 良好 | 8-10 分 | 切题，表达思想清楚，文字连贯 |
| 及格 | 5-7 分 | 基本切题，表达不够清楚，连贯性差 |
| 不及格 | 2-4 分 | 基本切题，但表达不清楚 |
| 差 | 1-2 分 | 条理不清，思路紊乱 |

### 4.2 四级与六级区别

| 维度 | 四级侧重点 | 六级侧重点 |
|------|-----------|-----------|
| 评分权重 | 格式、基础语法 | 逻辑深度、论证力度 |
| 纠错重点 | 格式规范、语法正确性 | 句式复杂度、词汇高级度 |
| 提分方向 | 规避基础错误 | 优化逻辑、升级句式 |

### 4.3 AI 模型

使用 **DeepSeek API**（`deepseek-chat` 模型），通过 Python FastAPI 服务中转调用。

## 8. 非功能性需求

### 8.1 性能

- 页面首屏加载 < 3 秒
- AI 批改响应 < 30 秒（取决于 DeepSeek API）

### 8.2 安全

- 密码 bcrypt 加密存储
- JWT 认证，区分用户/管理员权限
- 全部 API 参数化查询（防 SQL 注入）
- CORS 白名单控制

### 8.3 兼容性

- 主流浏览器（Chrome、Firefox、Safari、Edge）
- 响应式设计，桌面端优先
